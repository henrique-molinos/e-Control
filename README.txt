1. Instalar Python

2. Instalar as bibliotecas de cliente da API Google via CMD

	pip install --upgrade google-api-python-client google-auth-httplib2 google-auth-oauthlib

3. Copiar o diretório inteiro para a sua máquina

4. Certificar que você possui permissão de acesso junto ao desenvolvimento

5. O arquivo de execução é o main.py

	Ao abrir o arquivo de execução e selecionar cliente e loja, o programa abrirá o navegador para autenticação.

6. Após autenticar com a conta Google autorizada junto ao desenvolvimento, será criado um arquivo token no diretório de execução para que não seja necessário autenticar novamente.
